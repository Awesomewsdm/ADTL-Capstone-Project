# Assets

Drop your assets in this folder. The site expects these filenames (replace placeholders, or rename in `index.html`):

| File            | What it is                             |
| --------------- | -------------------------------------- |
| `favicon.png`   | Browser tab icon (32×32 or 64×64)      |
| `avatar.jpg`    | Hero portrait photo (square, ~600×600) |
| `project-1.jpg` | Project 1 cover image (16:10)          |
| `project-2.jpg` | Project 2 cover image (16:10)          |
| `project-3.jpg` | Project 3 cover image (16:10)          |
| `cv.pdf`        | Downloadable CV / résumé               |

Tip: compress images at [squoosh.app](https://squoosh.app) before adding — keeps the site fast.
